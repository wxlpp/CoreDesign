# Task: SnapshotTest target + 生成首版快照

## Description

在 App project 内创建第二个 target（XCTest），继承 SnapshotPreviews 的 `SnapshotTest` 基类以自动收编所有 `#Preview` 宏生成 PNG 快照。配置输出路径到 `docs/snapshots/`，运行首版全量快照。

## Acceptance Criteria
- [ ] XCTest target 存在，链接 `SnapshottingTests` library
- [ ] 测试类继承 `SnapshotTest`，无需覆写方法（收集所有 `#Preview`）
- [ ] `xcodebuild test` 通过，`docs/snapshots/` 下生成 ≥ 13 张 PNG（每组件至少 1 张）和 JSON metadata
- [ ] 环境变量 `TEST_RUNNER_SNAPSHOTS_EXPORT_DIR` 可指定输出目录
- [ ] 排除了非组件 preview（如 Toast 内部 helper preview、专注 ring / modifier preview 等）

## Technical Details

### Test Target 结构

```
App/
└── Tests/
    └── SnapshotTests.swift
```

```swift
import SnapshottingTests
import XCTest

final class SnapshotTests: SnapshotTest {
    // 默认收集所有 #Preview 宏，无需覆写
    // 排除非组件 preview 可覆写 excludedSnapshotPreviews()
    override func excludedSnapshotPreviews() -> [String] {
        [
            "ToastPreviewHarness",    // Toast 内部 helper
            "SearchFieldPreviewHost", // SearchField 内部 helper
            "BadgePreviewMatrix",     // 组合 preview 单独处理
            "ListRowPreviewGallery",
            "TagPreview",             // 排除 #Preview 宏里的组合视图
        ].flatMap { ["\($0)_Light", "\($0)_Dark"] }
    }
}
```

### 运行命令

```bash
TEST_RUNNER_SNAPSHOTS_EXPORT_DIR=docs/snapshots \
xcodebuild test \
  -project App/CoreDesignPreview.xcodeproj \
  -scheme SnapshotTests \
  -destination 'platform=iOS Simulator,name=iPhone 16' \
  -quiet
```

### 输出

```
docs/snapshots/
├── Badge_light.png
├── Badge_dark.png
├── Tag_light.png
├── Tag_dark.png
├── Banner_light.png
├── Banner_dark.png
├── ...（≥ 26 张，每组件 light + dark）
└── snapshot_manifest.json（SnapshotPreviews 自动生成）
```
