# Task: 语义色补全 + Color.focusRing 重命名为 borderFocus

## Description

扩展 `BorderColors.swift` / `SurfaceColors.swift` / `ContentColors.swift` 三个文件，新增 14 个 Primer 对齐的语义色 token；删除 `Color.focusRing` 并新增 `Color.borderFocus`；在 `Resources.xcassets/canvas/` 与 `Resources.xcassets/border/` 顶级目录下新增必要 colorset。

**本任务包含 epic 唯一破坏性变更**（删除 `Color.focusRing`），且是**关键路径瓶颈**（同时阻塞 Task 6 / 7 / 8）。单 agent 执行时应优先于 Task 2 / 3 / 9 启动。

## Acceptance Criteria

### 颜色 token 新增（共 14 个）

- [ ] `SurfaceColors.swift` 新增 6 个：`surfaceCanvas` / `surfaceCanvasSubtle` / `surfaceCanvasInset` / `surfacePanel` / `surfaceSidebar` / `surfaceCard`
- [ ] `BorderColors.swift` 新增 5 个：`borderMuted` / `borderHover` / `borderFocus` / `borderSelected` / `borderEmphasis`
- [ ] `ContentColors.swift` 新增 3 个：`contentMuted` / `contentSubtle` / `contentOnEmphasis`
- [ ] 每个新增 token 必须带 `///` doc-comment，标注对应的 Primer 概念名 + 与现有 token 的语义映射（示例：`/// Primer canvas.default. 接近现有 surfaceBase, 新代码优先使用本 token`）

### Color.focusRing 重命名

- [ ] `BorderColors.swift:24` 的 `public static var focusRing: Color = .accent` **删除**（不保留别名，PRD 已确认无下游依赖）

- [ ] 仓库内 `Color.focusRing` 引用已检查并修复，验证：`grep -rn 'Color\.focusRing\|\.focusRing\b' Sources/CoreDesign/ | grep -v 'View\.focusRing\|focusRing(' | wc -l` == 0

### xcassets

- [ ] 创建 `Resources.xcassets/border/` 顶级目录
- [ ] 在上述目录下新增必要的 colorset：至少 `canvas-default.colorset` / `canvas-subtle.colorset` / `canvas-inset.colorset`（surface-canvas 系列），以及 `border-focus.colorset`
- [ ] 其余 surface / border / content token 可复用 `ColorGrade` 中的 grey / brand 等已有色相，不强制新建 colorset

## Technical Details

- Primer color tokens 见 `https://github.com/primer/primitives/tree/<locked-version>/src/tokens/functional/color`
- `surfacePanel` / `surfaceSidebar` / `surfaceCard` 多数可不新建 colorset，直接基于现有 grey 系列或 system color 桥接（`secondarySystemBackground` 等）
- `borderMuted` / `borderHover` / `borderSelected` / `borderEmphasis` 多数可基于现有 grey/brand 系列
- `contentMuted ≈ contentTertiary` / `contentSubtle ≈ contentQuaternary`（在 doc-comment 中标注；具体取值是否完全等同由 Primer 定义裁决）
- 顶级目录命名（`canvas/`、`border/`）按 epic Architecture Decision #7
