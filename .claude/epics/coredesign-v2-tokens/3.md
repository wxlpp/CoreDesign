# Task: CoreTypography token

## Description

新增 `Sources/CoreDesign/Tokens/CoreTypography.swift`，提供 7+ 档对齐 Primer text styles 的 SwiftUI Font 命名，每档同时暴露推荐 lineSpacing 与 tracking。

## Acceptance Criteria

- [ ] 新建 `CoreTypography.swift`，caseless `public enum CoreTypography`
- [ ] 至少 7 档命名：建议 `displayLarge` / `titleLarge` / `titleMedium` / `bodyLarge` / `bodyMedium` / `bodySmall` / `caption`
- [ ] 每档暴露三件套：`public static let *Font: Font`、`public static let *LineSpacing: CGFloat`、`public static let *Tracking: CGFloat`
- [ ] `lineSpacing` 取值用公式 `max(0, primerLineHeight - fontSize)` 计算，公式以注释形式落到代码（不仅仅是 doc-comment 提及）
- [ ] 文件顶部含注释 `// Source of truth: docs/PRIMER_VERSION.md`
- [ ] 每档命名上方 `///` doc-comment 标注对应 Primer text style，并显式说明：**单行文本（按钮 label / 列表行）不会观察到 lineSpacing 效果，这是预期行为，不视为缺陷**

## Technical Details

- Primer typography token 见 `https://github.com/primer/primitives/tree/<locked-version>/src/tokens/functional/typography`
- 调用方写法：`.font(CoreTypography.bodyMediumFont).lineSpacing(CoreTypography.bodyMediumLineSpacing).tracking(CoreTypography.bodyMediumTracking)`
- letter-spacing → SwiftUI `tracking`；line-height → `lineSpacing` 公式如上
- 不需要自定义 `Font.Descriptor`；SwiftUI 内置 Font + lineSpacing/tracking 三件套足够覆盖 Primer typography
