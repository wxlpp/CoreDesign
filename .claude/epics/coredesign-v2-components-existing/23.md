# Task: SegmentedControl 重构

## Description

把 `Sources/CoreDesign/Components/SegmentedControl/SegmentedControl.swift` 按 v2-tokens 重构。当前文件 (~88 行) 含多处魔法数字：外框 `cornerRadius: 9`、thumb `cornerRadius: 7`、`height: 32`、thumb 阴影 `radius: 2, y: 1` 等。**不使用 `.glassEffect`**。

## Acceptance Criteria

### Token 化
- [ ] 外框 `RoundedRectangle(cornerRadius: 9)` → `CoreRadius.medium`（值 6，原 9 是 segmented 装饰圆角；与 Primer 对齐取 6）；如视觉差异显著，可改 `CoreRadius.large`（值 12），决策记入 PR description
- [ ] thumb `RoundedRectangle(cornerRadius: 7)` → `CoreRadius.small`（值 3）或 `CoreRadius.medium`（值 6），按视觉决定
- [ ] `frame(height: 32)` → `CoreControlMetrics.height(for: .regular)`（32pt）
- [ ] thumb 阴影 `Color.black.opacity(0.08), radius: 2, x: 0, y: 1` → 通过 `View.coreShadow(.small)` 应用
- [ ] padding（`.padding(2)` 等）→ `CoreSpacing.xxs`（2pt）
- [ ] `spacing: 2` / `spacing: 4` → `CoreSpacing.xxs` / `xs`
- [ ] 字号 `.font(.subheadline.weight(...))` → `CoreTypography.bodyMediumFont`（视觉抽查后决定 weight 调整方式）

### Glass 策略
