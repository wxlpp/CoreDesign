# Task: Primer 版本锁定 + 标量 token (Spacing/Radius/BorderWidth)

## Description

创建 `Sources/CoreDesign/Tokens/` 目录与 Primer 版本锚定文件 `PRIMER_VERSION.md`。新增三个最简单、零依赖的标量 token 文件：`CoreSpacing.swift`、`CoreRadius.swift`、`CoreBorderWidth.swift`。本任务是后续所有 token 任务（2/3/4/5/9）的依赖，必须先合并。

## Acceptance Criteria

- [ ] 新建目录 `Sources/CoreDesign/Tokens/`
- [ ] 新建 `docs/PRIMER_VERSION.md`，含字段：锁定的 Primer Primitives 版本号（git tag 或 npm release）、对应 GitHub ref 链接、锁定日期 ISO 8601、决策来源（PRD 路径）
- [ ] 新建 `CoreSpacing.swift`：caseless `public enum CoreSpacing`，10 个 `public static let: CGFloat`（none=0, xs=4, sm=8, md=12, lg=16, xl=24, xxl=32, xxxl=40, xxxxl=48, huge=64）
- [ ] 新建 `CoreRadius.swift`：caseless `public enum CoreRadius`，6 个 `public static let: CGFloat`（none=0, small=4, medium=6, large=8, xlarge=12, full=9999）
- [ ] 新建 `CoreBorderWidth.swift`：caseless `public enum CoreBorderWidth`，4 个 `public static let: CGFloat`（none=0, hairline=0.5, thin=1, thick=2）
- [ ] 三个 token 文件顶部均含注释 `// Source of truth: docs/PRIMER_VERSION.md`，**不重复声明版本字符串**
- [ ] 每个 token 的具体取值在 doc-comment 中标注对应的 Primer Primitives 概念（如 `// Primer space.3 ≈ 12pt`）

## Technical Details

- `Package.swift` 不需要改（`Sources/CoreDesign/` 通配生效，新增 Tokens/ 子目录自动纳入 target）
- 所有 token 类型必须 caseless enum + `public static let`，**不要用 raw-value enum**（避免调用方 `.rawValue`，且不能直接传给 `cornerRadius:` 等 SwiftUI 参数）
- 取值参考：动手前先查 `https://github.com/primer/primitives` 最新 stable release 的 `tokens/functional/size/size-coarse.json` 等文件，记录到 `PRIMER_VERSION.md`
- `PRIMER_VERSION.md` 是后续 token 任务的唯一权威来源，不可遗漏

## Dependencies

无前置任务。本任务是其他 token 任务（2/3/4/5/9）的前置。
