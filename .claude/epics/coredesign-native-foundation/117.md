# Task: 删除 6 个 GitHub 专用组件

## Description

删除 `BookCover`、`RefPill`、`StatusRow`、`EventRow`、`CommentCard`、`TimelineItem`——这六个组件共同服务于「GitHub Issue 时间线」一个场景，在通用设计系统里是死重，且它们的实现是 Primer 观感的集中体现（低圆角、无材质、web 密度）。
