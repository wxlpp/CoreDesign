# Task: 重铸字体 / 圆角 / 尺寸 / 阴影 token

## Description

把 `Tokens/` 下的度量类 token 从 Primer 标度换成 Apple HIG 标度。本任务只改 token **定义**，不动组件调用点——调用点迁移是 Task 005，拆开是为了让「新档位定案」与「逐点确认新档位用对了」成为两次独立的判断。

## Acceptance Criteria

### 字体（`CoreTypography.swift`）

- [ ] 12 档改为 Apple 文本样式语义并直接取系统文本样式：
      `largeTitle`→`.largeTitle` / `title`→`.title` / `title2`→`.title2` / `title3`→`.title3` /
      `headline`→`.headline` / `body`→`.body` / `callout`→`.callout` / `subheadline`→`.subheadline` /
      `footnote`→`.footnote` / `caption`→`.caption` / `captionMono`→`.caption` monospaced / `caption2`→`.caption2`
- [ ] 删除 `@ScaledMetric` 缩放机制、全部 `*LineSpacing` / `*Tracking` 常量、`Spec.scales` 开关、`Token.fixedFont`
- [ ] `.coreFont(_:)` 调用形态保留（它是全库统一文字入口，形态变更会波及每个组件文件）
- [ ] `grep -n "Font.system(size:" Sources/CoreDesign/Tokens/CoreTypography.swift` 返回 0 行

### 圆角（`CoreRadius.swift` + 新增 shape helper）

- [ ] `none 0 / small 6 / medium 10 / large 16 / xLarge 22`；删除 `smallPlus`(4) 与 `mediumPlus`(8)
- [ ] 新增统一 shape helper，内部固定 `style: .continuous`；组件不再直接构造 `RoundedRectangle`
- [ ] 提供 `ConcentricRectangle` 的使用约定与入口（嵌套于已知容器的元素用它，容器侧按需 `.containerShape(_:)`）

### 控件尺寸（`CoreControlMetrics.swift`）

- [ ] 高度改为 `mini 28 / small 32 / regular 44 / large 50 / extraLarge 56`
- [ ] 纵横 padding 相应调整
- [ ] 删除 `primerVerticalPadding(for:)` 逃生口

### 阴影（`CoreElevation.swift`）

- [ ] resting 档保持近乎平坦，层级交给 material + separator；`.large` 保留给真正的浮层
- [ ] 文档注释中的 Primer 考据替换为 Apple HIG 依据

### 通用

- [ ] 各 token 文件头去掉 `Source of truth: docs/PRIMER_VERSION.md`
- [ ] `swift build` 通过（组件调用点的适配由 Task 005 承担，本任务允许留下需 005 处理的编译错误清单，但须显式记录）

## Technical Details

**`.continuous` 必须经 helper 强制**：只改 `CoreRadius` 数值拿不到 Apple 观感，因为角样式要在每个 `RoundedRectangle` 构造点显式指定，漏一处就是一个风格不一致的元素。因此验收判据是「grep 裸 `RoundedRectangle` 为 0」（Task 005 达成），而本任务负责提供那个唯一出口。

**同名换值预警**：新 `small = 6` 恰等于旧 `medium`，`regular` 高度 32 → 44。这些对调用点编译零感知，视觉全变——Task 005 须逐点确认，本任务须在交付说明中列出全部换值项供其使用。
