# Task: 删除 Blossom trait 与 CoreGradient

## Description

移除 Blossom 主题能力。「强品牌色渐变皮肤」与「跟随系统外观」是两个方向的产品目标，同时维护会让后续每个 token 改动与新组件的验证矩阵翻倍。

`CoreGradient` 随之删除：它存在的唯一理由是让 Blossom 拥有渐变而默认主题退化为纯色，Blossom 移除后三个 token 退化成「纯色包一层 `AnyShapeStyle`」，是净负担。

本任务放在 token 重铸之前：`#if Blossom` 分流恰好压在 `ColorGrade` / `SurfaceColors` / `InteractionColors` 上，正是 Task 003 / 004 要改的文件，先删可避免在将被删除的分支上做迁移。
