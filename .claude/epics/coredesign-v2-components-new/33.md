# Task: SearchField

## Description

新增 GitHub 风格搜索输入框 `SearchField`，含 magnifying-glass 前缀 icon + clear button + focus ring 集成。

## Acceptance Criteria

### API 形态
- [ ] 新建 `Sources/CoreDesign/Components/SearchField/SearchField.swift`
- [ ] `public struct SearchField: View`
- [ ] `public init(text: Binding<String>, placeholder: String = "Search", onSubmit: ((String) -> Void)? = nil)`
- [ ] 内部用 `@FocusState private var isFocused: Bool` 跟踪焦点
- [ ] 内部用 SwiftUI `TextField` + 自定义 styling

### 视觉
- [ ] 容器：`Color.surfaceCanvasInset` 背景 + `Color.borderMuted` 边框 + `CoreRadius.medium` 圆角；focus 时 border 切换为 `Color.borderFocus` 或调用 `.focusRing(visible: self.isFocused)`
- [ ] 高度：`CoreControlMetrics.height(for: .regular)`（用 `frame(minHeight:)` 而非 `frame(height:)`，per PRD FR-B-3 + CoreControlMetrics doc-comment 推荐）
- [ ] 字号：`CoreTypography.bodyMediumFont`
- [ ] padding：`CoreControlMetrics.horizontalPadding(for: .regular)` + `CoreControlMetrics.verticalPadding(for: .regular)`
- [ ] 前缀 icon：`Image(systemName: "magnifyingglass")` foregroundStyle `Color.contentMuted`，size `CoreControlMetrics.iconSize(for: .regular)`
- [ ] clear button（仅 `text.isEmpty == false` 时显示）：`Image(systemName: "xmark.circle.fill")` foregroundStyle `Color.contentMuted`，点击清空 text + resign focus

### .focusRing 集成
- [ ] 用 `View.focusRing(visible: self.isFocused, color: .borderFocus, width: CoreBorderWidth.thick, cornerRadius: CoreRadius.medium)`
- [ ] **跨平台行为说明**（per epic risk + ADR）：iOS 与 macOS 共享同一套 SwiftUI overlay 实现（不是临时 fallback 分支）；macOS 不被 Accessibility Inspector 识别为系统 focus indicator，仅是视觉等价；这是 v2-tokens issue #9 spike 后的最终决议，本 task 引用即可，无需重新论证。doc-comment 内引述这条限制

### Submit 行为
- [ ] 按 Return / Enter 触发 `onSubmit?(text)`（用 SwiftUI `.onSubmit { ... }`）

### Token-clean
- [ ] `SearchField.swift` production 代码零 magic-numbers

### Build / test / preview
- [ ] `swift build -Xswiftc -warnings-as-errors` 双平台通过
- [ ] `swift test` 退出 0
- [ ] `#Preview` light / dark 双模式 + focused / unfocused / 含文字 / 空文本（clear button 隐藏 / 显示）共 8 种态视觉合理

### PR description 5 字段
- [ ] 5 字段全部填写
- [ ] doc-comment 完整性勾选（含 macOS focus ring 限制引用）

## Technical Details

参考 epic.md `### 输入 / 列表 / 侧栏`。`focusRing` 是已 merged token modifier，调用即可，**不**重新实现 NSFocusRing 或自定义 stroke。

`onSubmit: ((String) -> Void)?` 用 closure 而非 `@escaping () -> Void` 单独 var，与 SwiftUI Apple-native 风格一致。
