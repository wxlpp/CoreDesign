## Technical Details

参考 epic.md `### 输入 / 列表 / 侧栏`。本组件在视觉构成上有一处特别约束：accent 条不是 `Rectangle` 字面 2pt 宽度，而是 `Rectangle().frame(width: CoreBorderWidth.thick)`——避免直接 inline。

如视觉发现 2pt accent 条偏窄（譬如 GitHub 桌面客户端实际是 3-4pt），可改用 `CoreBorderWidth.thicker`（4pt）；记入 PR description。
