# Task: Badge（canary）

## Description

新增 GitHub 风格状态指示器 `Badge`。本 task 是 epic canary——最简单的新组件，验证 `BadgeVariant` 枚举参数化模式 + `surfaceCanvasSubtle` / `borderMuted` 颜色 token 在新建组件中的人体工程学。

**职责边界**（与 Tag 区分）：Badge 是**状态指示器**（5 固定 BadgeVariant level，颜色由 token 决定）。**负例**：不得接收 caller-defined palette（不得有 `Badge(color: Color)` overload）。

## Acceptance Criteria

### API 形态
- [ ] 新建 `Sources/CoreDesign/Components/Badge/Badge.swift`
- [ ] `public struct Badge<Label: View>: View`
- [ ] `public enum BadgeVariant { case info, success, warning, danger, neutral }` (Sendable)
- [ ] `public init(variant: BadgeVariant = .neutral, outlined: Bool = false, @ViewBuilder label: () -> Label)`

## Technical Details

参考 epic.md `### 简单结构组件`。本组件没有 `BadgeStyle` 协议（不像 Banner），是 parametric struct + enum——更轻量、更贴 Primer Label 形态。

可暴露 internal helper（`private static func backgroundColor(for: BadgeVariant) -> Color` 等）；这些 helper 不算 public API 表面，无 doc-comment 治理要求。
