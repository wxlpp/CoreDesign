# Task: EmptyState

## Description

新增 GitHub 风格空状态视图 `EmptyState`，用于"无数据 / 加载错误 / 待初始化"等场景的居中提示。

## Acceptance Criteria

### API 形态
- [ ] 新建 `Sources/CoreDesign/Components/EmptyState/EmptyState.swift`
- [ ] `public struct EmptyState<Action: View>: View`
- [ ] `public init(icon: Image, title: String, description: String? = nil, @ViewBuilder action: () -> Action)`
- [ ] `EmptyState(icon: Image, title: String, description: String?)` convenience init `where Action == EmptyView`
- [ ] `EmptyState(systemName: String, title: String, description: String?, ...)` convenience init 接 `Image(systemName:)`

### 视觉
- [ ] 居中布局：垂直 `VStack` of icon → title → description → optional action
- [ ] icon 尺寸：默认 `CoreSpacing.xxxxl` (48pt) 或 `CoreSpacing.huge` (64pt)；可加 `iconSize: CGFloat = CoreSpacing.xxxxl` 默认参数让调用方覆盖
- [ ] icon foregroundStyle：`Color.contentMuted`
- [ ] title 字号：`CoreTypography.titleMediumFont`，foregroundStyle `Color.contentPrimary`
- [ ] description 字号：`CoreTypography.bodyMediumFont`，foregroundStyle `Color.contentMuted`
- [ ] icon ↔ title 间距：`CoreSpacing.lg`；title ↔ description 间距：`CoreSpacing.xs`；description ↔ action 间距：`CoreSpacing.xl`
- [ ] 容器 padding：`CoreSpacing.xl`
