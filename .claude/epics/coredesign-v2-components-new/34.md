# Task: ListRow

## Description

新增 GitHub 风格列表行 `ListRow`：三块布局 leading（icon / avatar）+ label（标题 + 副标题）+ trailing（chevron / accessory）。三泛型 + EmptyView convenience init。

## Acceptance Criteria

### API 形态（per epic ADR #15 init 形态约束）
- [ ] 新建 `Sources/CoreDesign/Components/ListRow/ListRow.swift`
- [ ] `public struct ListRow<Leading: View, Trailing: View, Label: View>: View`
- [ ] **Designated init 全标签**：`public init(@ViewBuilder leading: () -> Leading, @ViewBuilder label: () -> Label, @ViewBuilder trailing: () -> Trailing)`——三个标签每个对应一个 `@ViewBuilder` 闭包
- [ ] **Convenience inits 只补缺省槽位**（不引入无标签闭包重载）：
  - `public init(@ViewBuilder label: () -> Label, @ViewBuilder trailing: () -> Trailing)` `where Leading == EmptyView`
  - `public init(@ViewBuilder leading: () -> Leading, @ViewBuilder label: () -> Label)` `where Trailing == EmptyView`
  - `public init(@ViewBuilder label: () -> Label)` `where Leading == EmptyView, Trailing == EmptyView`

### 视觉
- [ ] 默认背景：`View.surface(.canvas)`（无 hover 时透明 / canvas 色，按 `surface(_:)` 实现）
- [ ] hover 态：`Color.surfaceCanvasSubtle`（**用 SwiftUI `.onHover(perform:)` 跟踪**——SwiftUI 无原生 hover state binding，自管 `@State private var isHovered = false`）
- [ ] label 区：`VStack(alignment: .leading)` of 标题 (`CoreTypography.bodyMediumFont`, `Color.contentPrimary`) + 副标题 (`CoreTypography.bodySmallFont`, `Color.contentMuted`)
- [ ] leading / trailing icon：尺寸 `CoreControlMetrics.iconSize(for: .regular)`
- [ ] 高度：`frame(minHeight: CoreControlMetrics.height(for: .regular))`（不固定 height，让内容驱动；per CoreControlMetrics doc-comment 推荐）
- [ ] padding：`CoreControlMetrics.horizontalPadding(for: .regular)` + `CoreControlMetrics.verticalPadding(for: .regular)`
- [ ] leading ↔ label / label ↔ trailing 间距：`CoreSpacing.md`

### Hover token debt（**doc-comment 用同一段固定文字**，per epic ADR #2 跨任务 guardrail）
