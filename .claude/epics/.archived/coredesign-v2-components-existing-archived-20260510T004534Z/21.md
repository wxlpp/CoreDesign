# Task: Button styles 重构（canary）

## Description

把 `Sources/CoreDesign/Components/Button/styles/` 下 4 个 `ButtonStyle` 文件按 v2-tokens 重构去除魔法数字、按 Primer 视觉语言收齐，并**顺带复审** `Button/ButtonRoleStyleRole.swift` 的颜色映射（不为它单独起 task）。本 task 是 epic canary——它是组件库使用最广的一组，落地后验证 v2-tokens 在真实组件中的人体工程学。

## Acceptance Criteria

### Token 化（4 文件全部）
- [ ] `SolidButtonStyle.swift`：EdgeInsets 表 → `CoreControlMetrics.{horizontalPadding,verticalPadding}(for:)`；`textFont` → `CoreControlMetrics.font(for:)`；`cornerRadius` → `CoreRadius.full`（保留 Capsule 形态）；阴影按 Primer 实色按钮配 `CoreElevation.small` 或不带阴影
- [ ] `LightButtonStyle.swift`：同上的 token 化；阴影继续走 `CoreElevation.spec(for:)` 或 `View.coreShadow(_:)`
- [ ] `BorderlessButtonStyle.swift`：EdgeInsets 表 token 化；无视觉容器
- [ ] `CircularGlassButtonStyle.swift`：EdgeInsets / size token 化；**保留 `.glassEffect`**（白名单文件）

### Glass 策略（按 epic ADR #3）
- [ ] `SolidButtonStyle`：**去 `.glassEffect`**，按 Primer 实色按钮（背景实色 + 描边）
- [ ] `LightButtonStyle`：亮色 colorScheme 去 glass + 改柔和阴影；暗色 colorScheme 保留 glass
- [ ] `CircularGlassButtonStyle`：保留 glass（命名即语义）

### ButtonRoleStyleRole 顺带复审
- [ ] 检查 `ButtonRoleStyleRole.swift` 的 `color` / `activeColor` / `disabledColor` 映射是否仍与 v2 语义色（`.accent` / `.borderFocus` / `.warning` / `.danger` 等）对齐；如不对齐，更新映射
- [ ] **不**为 `ButtonRoleStyleRole.swift` 独立起子 task / 不要求它有 `#Preview`

### Magic-number 归零（按 PRD SC#1 + epic ADR #8）
- [ ] 4 个 ButtonStyle 文件，每文件下 PRD §SC#1 三条 grep 命中 = 0（排除 `#Preview`）

### Build / test / preview
- [ ] `swift build -Xswiftc -warnings-as-errors` (macOS) 通过 0 warning
- [ ] `swift build -Xswiftc -warnings-as-errors --triple arm64-apple-ios26.0-simulator` 通过 0 warning
- [ ] `swift test` 退出 0
- [ ] 4 个 Style 各自 `#Preview` 在 light / dark 双 colorScheme 下视觉合理

### PR description 5 字段（按 epic ADR #6）
- [ ] Light preview / Dark preview / 视觉变化摘要 / Glass 变化 / public API diff 全部填写
- [ ] doc-comment 完整性勾选确认（每个 public ButtonStyle struct + extension `ButtonStyle where Self == ...` 的 doc-comment 覆盖：使用场景、关键参数、Primer 概念对应、light/dark 行为差异）

## Technical Details

参考 epic.md `### Button 子系统`。关键 token 引用：

| 来源 | token |
|---|---|
| EdgeInsets 表 5 case | `CoreControlMetrics.horizontalPadding(for:)` + `verticalPadding(for:)` |
| `textFont` 5 case | `CoreControlMetrics.font(for:)` |
| Capsule 圆角 | `CoreRadius.full`（保持 Capsule 形态） |
| 描边宽度 0.8 | `CoreBorderWidth.hairline` (0.5) 或 `CoreBorderWidth.thin` (1)，按视觉决定 |
| 阴影参数 | `CoreElevation.small` / `.medium` 通过 `View.coreShadow(_:)` 应用 |
