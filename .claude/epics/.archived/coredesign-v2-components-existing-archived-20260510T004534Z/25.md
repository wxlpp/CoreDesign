- [ ] `LabelIcon`：外层 `font(.system(size: 26))` → `CoreControlMetrics.iconSize(for: .extraLarge)` + `CoreSpacing.xxl` (32pt)；内层 icon `font(.system(size: 16))` → `CoreControlMetrics.iconSize(for: .regular)` 或 `CoreSpacing.lg` (16pt)。**注意**：`CoreControlMetrics.iconSize(for:)` 已 token 化 icon 尺寸；优先使用它。

- [ ] `ChevronRightIcon`：保持简单；如未来需要尺寸控制，加默认参数走 `CoreControlMetrics.iconSize(for:)`
- [ ] 容器（如本 Form 内嵌 List / VStack 等）走 `View.surface(.panel)` 或对应 surface kind（Form 本身是 SwiftUI 原生 `Form`，不必动）
